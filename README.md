README – Week 5 Bootstrap Task (Hotel Management System)

Project Title

Hotel Management System Landing Page

Description

This project is a responsive landing page built using HTML and Bootstrap 5. It demonstrates how Bootstrap can be used to create a modern, mobile-friendly website with minimal custom CSS.

Features

- Responsive navigation bar with a hamburger menu.
- Hero section introducing the Hotel Management System.
- Three feature cards:
  - Room Booking
  - Reservation Management
  - Excellent Service
- Contact form for customer enquiries.
- Responsive layout using Bootstrap's grid system.
- Bootstrap utility classes for spacing and styling.

Technologies Used

- HTML5
- Bootstrap 5 (CDN)

Bootstrap Components Used

- Navbar
- Grid System
- Cards
- Buttons
- Forms
- Utility Classes

How to Run

1. Download or clone the project.
2. Open the "index.html" file in any modern web browser.
3. Ensure you have an internet connection so the Bootstrap CDN can load.
